# 🧮 Python Calculator

A simple command-line calculator built using Python.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous execution using loop
- Exit option

## 🛠️ Technologies Used
- Python
- VS Code
- Git & GitHub

## 📌 How to Run

1. Clone the repository
2. Open terminal in project folder
3. Run:

python calculator.py
